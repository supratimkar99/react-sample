## Installation and Setup

### Prequisites

* NodeJs- v14+
* Ruby- v2.7.2
* Ruby on Rails - v14.15.5
## How to run the project

* Git clone the repository
```
git clone [clone-link]
```
* Install dependencies
```
npm install
```
* Install gems
```
bundle install
```
* Migrate DB
```
rails db:migrate
```
* Seed data
```
rails db:seed
```
* Run the server
```
rails s
```
* Open Localhost server
```
http://localhost:3000
```
